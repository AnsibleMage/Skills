# VISUAL DENSITY v1.0 — 시각 밀도 규칙 모듈

> CREATIVE_GENERATION_FRAMEWORK 품질 모듈.
> "빈 공간" 없이 모든 영역이 의도적으로 채워져야 한다.
> 참조: Netflix, YouTube, Coursera, 세종대 도서관

---

## 원칙

```
1. 빈 공간은 "여백"이거나 "시각 콘텐츠"여야 한다. "채우지 못한 공간"은 금지.
2. 모든 카드/섹션에 최소 2개 이상의 시각 요소가 있어야 한다.
3. 데이터 포인트(숫자, 통계, 진행률)는 시각적으로 강조한다.
4. 장식 요소는 세계관에서 파생되어야 한다 (임의 장식 금지).
```

---

## 시각 밀도 7계층

### Layer 1: 배경 (Background Layer)

```
- 단색 배경 금지 (최소 미세한 그라디언트 또는 패턴)
- SLOT_1 메타포의 texture 규칙 적용
- 섹션 간 배경색 변화 (같은 색 2개 연속 금지)
```

```css
/* 미세 그라디언트 예시 — 순백이 아닌 방향성 있는 흰색 */
.section-bg {
  background: linear-gradient(180deg, #ffffff 0%, #f8f9fa 100%);
}
```

### Layer 2: 콘텐츠 컨테이너 (Container Layer)

```
- 카드에 그림자 또는 border 필수 (공중에 뜬 느낌 금지)
- 카드 배경색은 base-bg와 1~2단계 차이
- 카드 내부에 padding 20px+ (빽빽함 방지)
```

### Layer 3: 텍스트 계층 (Text Layer)

```
- 모든 카드에 최소 2줄 텍스트 (제목만 있는 카드 금지)
- 텍스트 크기 위계: 제목(16px+) > 설명(14px+) > 메타(14px)
- 메타 데이터 행: 최소 2개 정보 (예: "수강 1,234명 · 댓글 56")
```

### Layer 4: 시각 장치 (Visual Device Layer)

```
- 뱃지/태그: 카테고리 표시 (최소 1개/카드)
- 진행 바: 학습 진행률 시각화
- 아이콘/이니셜: 채널 아바타, 카테고리 표시
- 숫자 강조: 큰 폰트 + 다른 weight/color
```

### Layer 5: 인터랙션 힌트 (Interaction Hint Layer)

```
- "전체보기" 링크: 모든 섹션에 탐색 포인트
- 가로 스크롤 화살표: 스크롤 가능함을 표시
- hover 상태 변화: 인터랙션 가능함을 암시
- 포커스 링: 접근성 + 시각 피드백
```

### Layer 6: 장식 요소 (Decorative Layer)

```
- 세계관에서 파생된 장식만 허용
  - 건축사무소 → 격자 패턴, 도면 번호
  - 천문대 → 별 점 패턴, glow
  - 재봉공방 → dashed 선, stitch 패턴
- 임의 동그라미/삼각형 장식 금지
```

### Layer 7: 미세 디테일 (Micro Detail Layer)

```
- 구분선: 1px rgba 미세선
- 아이콘: ♡ 좋아요, 💬 댓글, 👤 수강생
- 뱃지 위치: 썸네일 좌상단 (카테고리), 우상단 (좋아요)
- 시간 표시: 우하단 (60분, 12:34)
```

## 섹션별 최소 밀도 기준

| 섹션 | 최소 시각 요소 | 필수 포함 |
|------|-------------|---------|
| 히어로/검색 | 3개 | 배경 그라디언트 + 검색바 + 검색 힌트 태그 |
| 학습현황 | 4개 | 도넛 차트 + 숫자 + 교육과목 pill + 진행률 |
| 코스 카드 | 5개/카드 | 썸네일 + 뱃지 + 제목 + 설명 + 메타데이터 |
| 이어서학습 | 4개/카드 | 썸네일 + 진행바 + 퍼센트 + 제목 |
| 채널 | 3개/카드 | 아바타 + 이름 + 구독자 수 |
| 도서 | 3개/카드 | 표지 + 제목 + 저자 |
| 연락처 | 2개/카드 | 아이콘/이모지 + 전화번호 |
| 푸터 | 3개 | 로고 + 링크 그룹 + 저작권 |

## "빈 느낌" 진단 규칙

```yaml
density_check:
  - [ ] 히어로에 배경 이미지/그라디언트가 있는가?
  - [ ] 모든 썸네일에 시각 콘텐츠가 있는가? (회색 사각형 0개)
  - [ ] 각 카드에 최소 2개 이상의 텍스트 레벨이 있는가?
  - [ ] 뱃지/태그가 카테고리 정보를 제공하는가?
  - [ ] 숫자 데이터(수강생, 진행률)가 시각적으로 강조되는가?
  - [ ] 섹션 간 배경색 변화가 있는가? (같은 색 2연속 금지)
  - [ ] 장식 요소가 세계관에서 파생되었는가? (임의 장식 아님)
  - [ ] 가로 스크롤 영역에 스크롤 인디케이터가 있는가?
  - [ ] 푸터가 3개 이상 정보 그룹으로 나뉘어 있는가?
```
