# 적용 기술스택 정의서 — 새로e아름 메인 페이지

## 개요

새로e아름 메인 페이지 리뉴얼에 적용할 기술스택을 정의한다. 기존 JSP 배포 환경을 유지하면서 모던 프론트엔드를 구현한다.

---

## 1. 기술스택 구성

### 1.1 스택 요약

| 계층 | 기술 | 버전 | 역할 | 도입 방식 |
|------|------|------|------|----------|
| **CSS** | Tailwind CSS | 3.x (Play CDN) | 유틸리티 기반 스타일링 | CDN 또는 로컬 |
| **JS (반응성)** | Alpine.js | 3.x | 데이터 바인딩, AJAX, 이벤트 | CDN 또는 로컬 |
| **JS (캐러셀)** | Swiper | 11.x | 배너 슬라이드, e-book Coverflow | CDN 또는 로컬 |
| **뷰 엔진** | JSP + JSTL | 기존 유지 | 서버사이드 렌더링, 세션 변수 | 기존 |
| **서버** | Spring MVC + Tomcat | 기존 유지 | 라우팅, API, 세션 | 기존 |

### 1.2 CDN 로드 코드

```html
<!-- Tailwind CSS -->
<script src="https://cdn.tailwindcss.com"></script>

<!-- Alpine.js -->
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

<!-- Swiper (캐러셀) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.css">
<script src="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.js"></script>
```

### 1.3 CDN 불가 시 로컬 배포

```
/lms/resources/new-main/
├── js/
│   ├── tailwindcss.js        ← cdn.tailwindcss.com에서 다운로드
│   ├── alpine.min.js         ← jsdelivr에서 다운로드 (14KB)
│   └── swiper-bundle.min.js  ← jsdelivr에서 다운로드

    └── swiper-bundle.min.css
```

```html
<link rel="stylesheet" href="/lms/resources/new-main/css/swiper-bundle.min.css">
<script src="/lms/resources/new-main/js/swiper-bundle.min.js"></script>
```

### 2.2 기존 CSS 충돌 방지

기존 `kpotilxp.css` + Bootstrap 3과 공존하기 위해, 새 메인 영역을 `#new-main`으로 스코프 제한:

```html
<script>
tailwind.config = {
    important: '#new-main',  // 이 ID 안에서만 Tailwind 우선 적용
    // ...theme 설정
}
</script>
```

헤더/풋터는 기존 CSS 그대로, 바디(`#new-main`) 안에서만 Tailwind 적용.

## 3. Alpine.js 사용 패턴

### 3.1 AJAX 데이터 로드

```html
<div x-data="{ items: [], loading: true }" x-init="
    fetch('/lms/portal/home/mainViewCourseListJson.ajax', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify({crsListId: 'crsRecmList'})
    })
    .then(r => r.json())
    .then(d => { items = d.crsList || []; loading = false; })
">
    <!-- 로딩 -->
    <div x-show="loading" class="animate-pulse bg-gray-100 h-48 rounded-xl"></div>

    <!-- 데이터 렌더링 -->
    <template x-for="item in items" :key="item.crsSessId">
        <div x-text="item.crsNm"></div>
    </template>
</div>
```

### 3.2 JSP 서버 변수 전달

```jsp
<div x-data='{
    user: {
        name: "<c:out value="${sessionScope.userName}" />",
        point: <c:out value="${sessionScope.arumPoint}" default="0" />
    }
}'>
    <span x-text="user.name"></span>
    <span x-text="user.point.toLocaleString()"></span>
</div>
```

### 3.3 탭 전환

```html
<div x-data="{ activeTab: 'must' }">
    <div class="flex gap-2 mb-4">
        <button @click="activeTab = 'must'"
                :class="activeTab === 'must' ? 'bg-arum-blue text-white' : 'bg-gray-100'"
                class="px-4 py-2 rounded-full text-sm transition">필수과정</button>
        <button @click="activeTab = 'recommend'"
                :class="activeTab === 'recommend' ? 'bg-arum-blue text-white' : 'bg-gray-100'"
                class="px-4 py-2 rounded-full text-sm transition">추천과정</button>
    </div>
    <div x-show="activeTab === 'must'" x-transition>필수과정 내용</div>
    <div x-show="activeTab === 'recommend'" x-transition>추천과정 내용</div>
</div>
```

### 3.4 주요 속성 레퍼런스

| 속성 | 역할 | 예시 |
|------|------|------|
| `x-data` | 데이터 정의 | `x-data="{ open: false }"` |
| `x-init` | 초기화 (AJAX 등) | `x-init="fetch(...).then(...)"` |
| `x-show` | 표시/숨김 | `x-show="open"` |
| `x-for` | 반복 | `x-for="item in items"` |
| `x-text` | 텍스트 바인딩 | `x-text="item.name"` |
| `:속성` | 속성 바인딩 | `:src="item.imgUrl"` |
| `@이벤트` | 이벤트 | `@click="open = !open"` |
| `x-transition` | 애니메이션 | `x-transition.duration.300ms` |

## 4. 사용할 AJAX 엔드포인트

기존 엔드포인트를 그대로 사용한다. 백엔드 수정 없음.

| 엔드포인트 | 메서드 | 용도 | 사용 섹션 |
|-----------|--------|------|----------|
| `/lms/portal/home/mainViewCourseListJson.ajax` | POST | 과정 목록 (필수/추천/인기/학습중) | T3, T4, T5 |
| `/lms/portal/home/mainViewEbookListJson.ajax` | POST | e-book 목록 | T7 |
| `/lms/portal/home/eduTime.ajax` | POST | 교육시간, 이수현황 | T2 |
| `/lms/portal/home/noticePopup.ajax` | POST | 팝업 공지 | 팝업 |
| `/lms/portal/home/mainViewConnnectLibrary.ajax` | POST | 전자도서관 SSO | T7 |

**과정 목록 crsListId 값**:

| 값 | 의미 | 섹션 |
|-----|------|------|
| `crsAppointList` | 필수과정 | T3 |
| `crsRecmList` | 추천과정 | T4 |
| `crsProgList` | 학습중 과정 | T5 |
| `crsHotList` | 인기과정 | (선택) |

```

    ├── t1_hero.jsp             ← 배너 + 퀵네비
```

## 7. 반응형 브레이크포인트

Tailwind 기본 브레이크포인트를 그대로 사용:

| 접두사 | 최소 너비 | 용도 |
|--------|----------|------|
| (없음) | 0px | 모바일 기본 |
| `sm:` | 640px | 모바일 가로 |
| `md:` | 768px | 태블릿 |
| `lg:` | 1024px | 데스크톱 |
| `xl:` | 1280px | 와이드 |

**카드 그리드 반응형 패턴**:
```html
<div class="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-5 gap-4">
```
- 모바일: 2열 → 태블릿: 3열 → PC: 5열
