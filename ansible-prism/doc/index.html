UX Notes

  • ① Hero search with large input field makes AI curator the focal point — "search-first" reduces cognitive load for new users
  • ② Quick status bar below search shows key metrics at a glance without leaving the search context

Summary

Optimization goal: Modernize

OptionApproachModernize Score
1: Classic StackExisting pattern — safe baseline★★☆☆☆
2: Dashboard HubAll metrics at top, data-dense overview★★★★☆
3: Progressive RevealAccordion sections, clean initial view★★★☆☆
4: Card Mosaic ★Modular card grid, responsive masonry★★★★★
5: Search PortalSearch-first with contextual discovery★★★★☆

Recommendation: Option 4 (Card Mosaic) — The modular card grid best serves the "modernize" goal. It eliminates horizontal carousels in favor of a scannable grid, naturally adapts to responsive breakpoints, and presents all content types (courses, channels, books, status) in a unified visual language. The masonry-style layout feels contemporary while maintaining all existing content and functionality.
