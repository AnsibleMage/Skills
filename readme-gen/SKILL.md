현재 폴더의 README.md 파일을 생성해주세요.

## 대상 폴더
$ARGUMENTS (폴더 경로 또는 현재 폴더)

## 수행 작업
1. 폴더 구조 분석 (파일/서브폴더 목록)
2. 주요 파일 내용 파악
3. README.md 작성:
   - 영어 버전 먼저 생성 (README.md)
   - 한국어 버전 생성 (README_KO.md)
4. 저장 완료 보고

## README 구조 (필수)

```markdown
# [Project/Folder Name]

> [One-line description]

## Overview
[프로젝트/폴더 설명 2-3문장]

## Structure
[폴더/파일 트리 구조]

## Key Components
[주요 구성요소 설명 테이블]

## Quick Start / Usage
[사용법 또는 시작 방법]

## Related
[관련 문서/링크]

---
*Generated by 🎵 Ari & 🔧 An*
```

## 작성 원칙
- GitHub 방문자가 읽는다고 가정
- 간결하고 명확하게
- 이모지 적절히 사용
- 폴더의 정체성과 목적 명확히 표현
